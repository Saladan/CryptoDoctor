CryptoDoctor

Einfache Ver- und Entschlüsselung von Texten aller Art

View project on GitHub

CryptoDoctor ist ein deutsches Tool, womit man sehr einfach Texte ver- und entschlüsseln kann. Es sind sogar mehrere Verschlüsselungen möglich, die in einer Liste angeordnet werden können und demzufolge gleichzeitig ausgeführt werden.


Features:
  • Cäsar-Verschlüsselung / Caesar-Cipher
  • Vigenère-Verschlüsselung / Vigenère-Cipher
  • Binäre Verschlüsselung (XOR) / Binary XOR Cipher

Coming Soon:
  • Mehr Verschlüsselungen / More Ciphers
  • Mehr Übersetzungen (Ab der Erarbeitung einer englischen Version wird auch dieses Dokument in englischer Sprache verfasst werden) / More Translations
  • Import- und Export-Funktion für Textdateien / Import and Export for text Files

Systemvoraussetzungen / System Requirements:
  • Java jre7 oder höher / Java jre7 or higher
